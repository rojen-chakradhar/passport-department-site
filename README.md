# Department of Passport | Nepal

## Overview
This project is a static informational website that explains the passport application process in Nepal in a clear, structured, and beginner-friendly way.

Official government websites often contain fragmented or hard-to-navigate information. This site consolidates that information into a single, easy-to-understand guide for first-time applicants, general public, renewal, minor, etc.

## Purpose & Impact
The goal of this project is to:
- Help users understand the passport application process before visiting government offices.
- Reduce confusion around requried documents, fee, and timeliness.
- Present public information in a more accessible and readable format.
This project focuses on public belefit, clarity, and accessibility, not monitization.

## Features
- Multi-page static website.
- Step-by-step explanation of the application process.
- Clear list of requried documents.
- Fees and processing time table.
- FAQ section addressing common concerns.
- Mobile-responsive layout.
- Simple, readable UI with accessibility in mind.
